# Bitcoin Block Analyzer

## Описание

Программа для анализа блоков Биткойна, которая:
1. Загружает блок по номеру из сети
2. Извлекает nonce из заголовка блока
3. Перебирает заданный диапазон значений nonce
4. Для каждого nonce вычисляет двойной SHA-256 с записью промежуточных состояний
5. Подсчитывает количество нулевых байт/слов в записанных состояниях
6. Строит автокорреляционную функцию полученного массива

## Требования

- C++17
- libcurl (для HTTP-запросов)
- nlohmann/json (для парсинга конфигурации)

### Windows

Пошаговая инструкция:
Установите vcpkg

Откройте Git Bash или любую консоль и перейдите в папку, где хотите установить vcpkg (например, C:\dev\).

Склонируйте репозиторий: git clone https://github.com/microsoft/vcpkg.git

Перейдите в папку: cd vcpkg

Запустите установку: .\bootstrap-vcpkg.bat

Подключите vcpkg к Visual Studio

Выполните команду: .\vcpkg integrate install

После этого Visual Studio автоматически "увидит" все библиотеки, установленные через vcpkg.

Установите libcurl

Выполните команду: .\vcpkg install curl:x64-windows (или curl:x86-windows для 32-битной версии).

vcpkg скачает исходники, скомпилирует их и установит в свою директорию.
